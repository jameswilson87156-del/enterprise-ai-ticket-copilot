# 每次发给 Codex 的单任务模板

复制下面整段，把方括号内容替换成当前任务。一次只提交一个任务。

```text
请在当前项目中先自行审查全项目现状，再执行本任务。

本轮唯一任务：
[粘贴 TODO.md 中一个任务的完整内容]

任务边界：
- 允许修改：[文件或目录]
- 不允许修改：[文件、功能或行为]
- 不在本轮处理：[后续功能]

执行要求：
1. 先读取 AGENTS.md、CLAUDE.md、相关 docs、TODO.md、HANDOFF.md 和相关源码。
2. 存在 Git 仓库时先查看 git status 和当前 diff，保留用户已有的无关改动。
3. 开始修改前，先复述项目现状、本轮目标、涉及文件、风险和验收方式。
4. 只做这个任务，不顺手实现下一个 TODO，不做无关重构。
5. 修改后运行实际可用且相关的测试、构建、lint、类型检查或启动验证。
6. 如果没有自动化测试，执行可重复的手工检查；不能验证的内容如实说明，不能编造通过。
7. 更新 TODO.md 和 HANDOFF.md，但不要覆盖历史记录。

完成后请输出：
- 修改文件列表，以及每个文件为什么修改；
- 实际执行的命令和原始结果摘要；
- 未执行的测试及原因；
- 按什么步骤验收；
- 剩余风险；
- 是否建议把当前 TODO 标记为完成。

完成本任务后停止，等待我确认，不要继续下一项。
```
